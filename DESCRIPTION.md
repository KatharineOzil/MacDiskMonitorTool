# 项目描述

**[English Version](DESCRIPTION_EN.md)** | **中文版本**

## 简短描述（一句话）
一套用于监控macOS磁盘空间使用情况的脚本工具，可追踪文件创建事件并记录详细信息，帮助用户快速定位磁盘空间消耗来源。

## 详细描述（一段话）
这是一个macOS磁盘空间监控脚本套装，包含三个不同版本的监控工具：原版脚本（基于fswatch事件驱动，CPU占用极低）、简化版脚本（无需任何依赖，使用find命令定期扫描）和Python版脚本（基于watchdog库，事件驱动且跨平台）。所有脚本都能实时监控文件创建事件，记录文件路径、大小和时间戳，生成CSV格式日志，帮助用户追踪开机后磁盘空间快速消耗的问题。项目提供了完整的文档、使用示例和日志分析技巧，适用于日常磁盘空间监控、应用程序行为分析和系统性能优化。

## 技术特点
- **多版本脚本**：适应不同环境和需求
- **事件驱动监控**：Python版和原版脚本CPU占用极低
- **日志记录**：CSV格式，便于后续分析和处理
- **跨平台潜力**：Python版可轻松适配其他操作系统
- **完整文档**：包含使用说明、故障排除和实际应用场景

## 适用场景
1. **磁盘空间问题诊断**：追踪开机后空间快速消耗的元凶
2. **应用程序行为监控**：分析特定软件的文件创建模式
3. **系统性能优化**：识别不必要的文件创建和临时文件堆积
4. **开发调试**：监控应用程序的文件系统操作
5. **安全审计**：检测异常的文件创建活动

## 使用示例
```bash
# 监控整个用户目录
python3 disk_monitor_python.py /Users

# 监控特定应用程序目录
python3 disk_monitor_python.py ~/Library/Application\ Support/Google/Chrome

# 生成带时间戳的日志文件
python3 disk_monitor_python.py /tmp ~/Desktop/monitor_$(date +%Y%m%d_%H%M%S).log
```
